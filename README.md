## Face Recognition with OpenCV
### To run:
Install dependencies:
```
pip install opencv
```
Run on static image:
```
python face.py
```
Sample:

![Pasted image](https://github.com/user-attachments/assets/f2235f3a-4e1c-43b3-a06f-89ea4d6de027)


Run live recognition:
```python
python facescan.py
```
